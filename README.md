# FUTURE_DS_02
This project focuses on analyzing Facebook and Instagram ad campaign data to evaluate marketing performance.
An interactive Power BI dashboard was created to track key advertising metrics and generate actionable insights
that help businesses optimize ad spend and improve Return on Investment (ROI).

Dashboard Highlights
- KPI Cards (Impressions, Clicks, Spend, CTR, ROI)
- Top Ads by Clicks
- Audience Analysis (Age & Gender)
- Time-based Performance Trends
- Interactive Filters

Tools Used
- Power BI
- DAX
- Excel
